# Basic Calculator in Python

A simple command-line calculator written in Python that performs basic arithmetic operations on two numbers.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Easy-to-understand beginner-friendly code

## How It Works

1. The program asks the user to enter two integers.
2. The user chooses an operation:

   * `add`
   * `sub`
   * `mul`
   * `div`
3. The calculator performs the selected operation and displays the result.

## Technologies Used

* Python 3

## How to Run

1. Clone this repository.
2. Open a terminal in the project directory.
3. Run the program using:

```bash
python calculator.py
```
